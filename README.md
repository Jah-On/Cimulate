# Cimulate
A C++ library to simulate key, mouse, and joystick actions. (Will come with Python lib as well)

Linux:
  - Mouse: Working
  - Keyboard: Working
  - Joystick: Not implemented

Windows:
  - Mouse: Not implemented
  - Keyboard: Not implemented
  - Joystick: Not implemented

X OSes:
  - Mouse: Not implemented
  - Keyboard: Not implemented
  - Joystick: Not implemented

Android:
  - Mouse: Not implemented
  - Keyboard: Not implemented
  - Joystick: Not implemented
